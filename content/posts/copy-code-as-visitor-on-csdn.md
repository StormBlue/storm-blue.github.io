### 方案一、开启 designMode

`Console`中输入`document.designMode='on'`

### 方案二、开启 contentEditable

`Console`中输入`document.body.contentEditable=true `，以 Ctrl+X 剪切内容

### 方案三、disable JavaScript

开启开发者工具，Ctrl-Shift-P，输入 JavaScript，选中 Disable JavaScript
